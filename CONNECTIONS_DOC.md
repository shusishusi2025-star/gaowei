## 3. 开发者无缝对接指南 (Developer Self-Host Manual)

本项目的 Express 与 React 对接严密，易于本地二次开发和直连部署：
1. **API Key 提供**：在设置管理窗口中，可以直接粘贴您自己的 `ApiKey` 进行本地临时写入测试；或者在项目根目录下，直接创建 `.env` 文件并填入：
   ```env
   GEMINI_API_KEY=您的真实GeminiKey
   ```
2. **本地模型挂载**：如需使用本地离线模型，启动您的本地 Ollama 服务，在模型设置中将 `本地端` 配置为 `http://localhost:11434`，一键连接即可以毫秒级零资费体验本地大模型（如 `qwen2.5` / `llama3`）对您店铺的数字员工全托管赋能。
